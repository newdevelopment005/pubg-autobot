# PUBG Autobot - How to Run

## Running the Bot
1. Make sure all dependencies are installed (see requirements.txt).
2. Ensure `pubg_custom_yolov8.pt` is in the same folder as `main.py`.
3. Run the bot:
   ```bash
   python main.py
   ```

## Controls
- `ESC`: Toggle human override (pause/resume bot).
- `F1`: Show/hide control panel.
- Tray icon allows you to reopen or quit the bot safely.

The GUI includes learning control and shows override status with flashing border when active.
